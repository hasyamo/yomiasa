## タイトル（確定）

**「キタコレだけレベルアップな件」**
- 元ネタ: アニメ「俺だけレベルアップな件」（俺レベ / Solo Leveling）。
- 三重のかかり: ①KITAcore の "core" ②KITAさんが名乗る「ログプレイヤー」 ③俺レベの「プレイヤー」。
- 原作（ウェブ小説）は2018年完結、ウェブトゥーン2021完結、アニメは第1期2024・第2期2025でクライマックス進行中。結末まで原作・漫画で完結しているので、オチを踏まえた演出（最終的にシャドウモナークに至る等）を安心して使える。

## 着想の流れ（記事の"作った過程"として使える）

1. もともとは「ワイ語可視化」だけの技術ネタ（KITACORE_MODE_PLAN.md の初期版）。KITAさんが一人称「ワイ」を使う関西弁キャラなのを可視化して驚かせる、が出発点。
2. トグルの置き方を相談 → ベタなトグルより「**プロフィールアイコンをダブルタップで金縁に覚醒**」という隠しコマンド案が出て採用。サプライズ機能に隠しコマンドが似合う。
3. 「ON/OFF時にシステムからメッセージが届く演出」→ 俺レベの「システム」（無機質な青いウィンドウ、`[ ]` 通知）を借りる方向に。
4. タイトル「キタコレだけレベルアップな件」が降ってきて、演出の軸が確定。以降「レベル」「ランク」「ボス」と発想が膨らむ。
5. 俺レベの階級（E〜S級＋国家級＋君主/シャドウモナーク）を称号に流用＝8段階。
6. 「進捗率でランクにすると新記事が出るたびに下がる」問題に気づく → **累計（積み上げ式）**に変えて解決。新記事が出ても下がらない＝「書き続ける限りキタコレが成長する終わらない物語」。
7. ボスキャラを出したくなる → 記事クイズで戦う案 → クイズは作るのが大変なので「**覚醒前の記事だけクイズ（有限）**」に限定。これがワイ語ゼロの黎明期記事に"読む価値"を与える解にもなった。
8. ボスの正体に悩んでいたら、KITAさん自作の記事「ヴィランを1体ずつ作っていたら『終焉の評議会』みたいな絵ができた話」(#379) のヴィラン6体がそのまま使えると判明。**自分が作ったボスに挑んで覚醒する**構造に。
9. 鍵システム（俺レベのダンジョン/鍵モチーフ）: 通常記事クイズ正解で鍵入手→鍵でボス挑戦→撃破でランクアップ→上位ボス解放。

## 実データで分かった事実（記事の"証拠"・グラフ素材）

KITAさん(ktcrs1107)の全記事本文をプロキシ経由で取得・解析（392記事、`.claude/kitacore_wai_counts.csv`）。

- **ワイ語の起点と覚醒**:
  - 初出は意外にも 2024-06-10「Spotify〜 #1」でワイ4回。だが**散発**で直後ゼロに戻る（黎明期）。
  - **真の覚醒は2025年10月**。前月まで使用率ほぼ0 → 10月にいきなり 23/38記事（61%）でワイ語112回に跳躍。
  - 11月以降は毎月200回前後、使用率9割で定着＝完全に「ワイ」キャラ確立。
- **累計**: ワイ 1,774 / 関西語セット 894 / 合計 2,668。
- ワイ使用記事 295/392（75%）。1記事最大ワイ数 = 24（2026-05-30「Threads再開〜ワイの世界観発掘記録」）。覚醒後の記事あたり平均ワイ5.3。
- 月別推移グラフ（黎明期ゼロ → 2025.10 で爆発）が、そのまま「キタコレ覚醒の瞬間」の可視化として記事に映える。
- 黎明期（〜2025.9）は約57記事で、ほぼ Python/技術記事。「ワイ」未確立の"覚醒前夜"。
